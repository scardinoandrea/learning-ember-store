## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/scardinoandrea/learning-ember-store.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Run the project
   ```sh
   ember s
   ```
